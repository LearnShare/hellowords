# C

## Tools

+ Editor: [Visual Studio Code](https://code.visualstudio.com/)
  + plugin: [C/C++](https://marketplace.visualstudio.com/items?itemName=ms-vscode.cpptools)
+ Compiler: [GCC](https://gcc.gnu.org/)

## Hello, World!

```c
#include <stdio.h>

int main(void) {
  printf("Hello, World!");
}
```

## Build

```bash
gcc helloworld.c -o helloworld.exe
```

## Run

```bash
helloworld
```
